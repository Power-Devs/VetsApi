# Vets Api

Uma api rest que provê informações ao VetsApp.

## Requisitos
- JDK 11

## Como rodar
É nessário ter a JDK 11 instalado para rodar a aplicação.

Abra o console dentro da pasta raíz do projeto e insira o comando a seguir:

```bash
mvnw spring-boot:run
 ```

## Tecnologias
- Java
- Spring Web
- Spring Boot
- Maven
